# Setup

- pip install requirements.txt
- create a secrets.py
  - Set `CLIENT_SECRET` inside to an [auth-token for your account](https://sentry.io/settings/account/api/auth-tokens/)
	- The token will need at least `org:read`
- to start `python discover.py`
